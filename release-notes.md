## Still Alive v1.2.0

### 更新日志

#### ✨ 新功能
- **自动打卡功能**：开启后可在解锁屏幕时自动完成每日签到
- **签到历史记录**：查看所有历史打卡记录
- **打卡状态刷新**：从后台恢复时自动刷新签到状态
- **每日签到通知**：每日定时提醒签到（可配置时间）

#### 🔧 改进
- 优化设置界面结构，新增「基本设置」「通知设置」「自动打卡设置」等独立页面
- 增强状态管理，使用 Repository 模式管理数据

#### 🐛 Bug 修复
- 修复某些情况下签到状态未及时更新的问题

### 下载说明
点击下方 **Assets** 中的 `app-release.apk` 下载安装包。

### 安装方法
1. 下载 APK 文件
2. 在 Android 设备上打开并安装
3. 如提示「允许安装未知应用」,请点击允许
